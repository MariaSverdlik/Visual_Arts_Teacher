# Portfolio site

Статический сайт-портфолио учителя рисования на HTML, CSS и JS.

## Как добавлять личные работы

1. Положите изображение в `images/works/`.
2. Добавьте запись в `data/works.json`.
3. Управляйте порядком через поле `order`: меньшее число показывается раньше.

Пример:

```json
{
  "id": "work-09",
  "title": "Новая работа",
  "year": "2026",
  "category": "Живопись",
  "technique": "Масло",
  "image": "images/works/work-09.jpg",
  "order": 15,
  "description": "Короткое описание работы."
}
```

## Как добавлять работы учеников

Используйте ту же схему, но изображения кладите в `images/students/`, а записи добавляйте в `data/students.json`.

## Изображения дизайна

Фоны, портреты и изображения для оформления сайта лежат в `images/site/`.

## Локальный просмотр

Так как галереи загружаются из JSON, сайт лучше открывать через локальный сервер:

```powershell
python -m http.server 8080
```

После запуска откройте `http://127.0.0.1:8080/`.
